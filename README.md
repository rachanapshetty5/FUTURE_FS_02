# FUTURE_FS_01 - Mini CRM

## Project Description

Mini CRM is a full-stack MERN application developed to manage customer leads. It allows users to add, view, update, and delete leads while storing data in MongoDB Atlas.

---

## Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Axios
- Tailwind CSS
- React Router

---

## Features

- Add Lead
- View Leads
- Edit Lead
- Delete Lead
- Dynamic Dashboard Statistics
- Responsive Dashboard

## Installation

### Clone Repository

```bash
git clone https://github.com/rachanapshetty5/FUTURE_FS_01.git
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd clientnpm
npm install
npm run dev
```
